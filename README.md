# Recruitment task - HexOcean

## Installation

1: Clone repository


```sh
git clone https://github.com/bsylwestrzak-dev/HexOceanTask.git
```
2: Generate node modules using npm inside project folder
```sh
npm install
```

3: Run the application
```sh
npm start
```
